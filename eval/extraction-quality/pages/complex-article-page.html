How Local-First Crawlers Stay Fast

Local-first crawlers win by avoiding browser work until content signals justify it.

Pipeline

The core loop is small:

  • Normalize URLs and strip tracking parameters.
  • Fetch with strict byte caps and timeouts.
  • Recover data islands before launching a browser.

Escalation policy

Use fast, then readable, then fingerprint, and only then browser. 

if (signals.sparseDom && signals.hasDataIsland) {
  return recoverFromJsonScripts(page)
}

Results

Mode Latency Quality
fast low good
browser high best for SPAs
