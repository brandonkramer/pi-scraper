Widget SDK Documentation

The Widget SDK lets agents crawl, map, and extract structured data from content systems.

Install

npm install @example/widget-sdk
node examples/quickstart.mjs

Quickstart

  1. Create a client with new WidgetClient().
  2. Call client.scrape(url) with a public HTTPS URL.
  3. Persist the returned responseId for later retrieval.

Tip: use mode: "fast" for static pages and escalate only when needed.

Options

Name Default Description
timeoutSeconds 30 Network timeout.
respectRobots true Honor robots.txt.
